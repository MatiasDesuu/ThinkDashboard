# Theme Examples

This document contains various theme configurations for ThinkDashboard. Each theme defines colors for text, backgrounds, borders, and accents.

To use them, just copy the corresponding value into the Custom Theme in `/colors` configuration.

## Eink

```json
{
    "name": "Eink",
    "textPrimary": "#000000",
    "textSecondary": "#000000",
    "textTertiary": "#000000",
    "backgroundPrimary": "#ffffff",
    "backgroundSecondary": "#ffffff",
    "backgroundDots": "#ffffff",
    "backgroundModal": "rgba(255, 255, 255, 1)",
    "borderPrimary": "#000000",
    "borderSecondary": "#000000",
    "accentSuccess": "#000000",
    "accentWarning": "#000000",
    "accentError": "#000000"
}
```
